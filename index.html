Tutorial

This project is a multi-page web client that provides information given by an API. It uses Bootstrap for responsive design and jQuery AJAX for calls to API.
The first part - Weather - is an application that receives a place and gives back the 5-day weather forecast for that location.
The second part - History - is a page where the user can enter a date and max number of lines and then receive a list of all weather requests made on that date.
